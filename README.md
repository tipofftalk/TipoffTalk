# TipoffTalk Vercel Build

## Deploy on Vercel
1. Go to Vercel.
2. Click **Add New Project**.
3. Upload this entire folder.
4. Vercel should detect **Next.js** automatically.
5. Click **Deploy**.

## Local run
```bash
npm install
npm run dev
```
